## Installation & Setup

Clone the repository:

```bash
git clone https://github.com/sanskritiigarg/Portfolio.git
```

Navigate into the project:
```
cd Portfolio
```

Install dependencies:
```
npm install
```

Run development server:
```
npm run dev
```

Open `http://localhost:3000`
